Summery

The Application was written in a Swift 5.3. The App represent Headlines API and Authentication. Application contains the three layers which are the Data Layer, Domain Layer or Business Layer and Presentation Layer in MVVM.

Requirements

  • Xcode 12.2 or laster.
  • Swift 5.3.1 or later.
  • iOS 13.0 or later.

Installation

  • Step 1: Download or clone the project github repository
    $ git clone https://github.com/farshadmb/YouTV-App
  • Step 2: Install Bundler via Running the below command Bash $ gem install bundler Install Cocoapods via Bash $ gem install cocoapods --user-install

then run $ pods install command.

  • Step 3: Open YouTVDemo.xcworkspace file.

  • Step 4: Run and enjoy the app.
